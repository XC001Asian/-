# Bank Transaction Auto-Classifier (銀行帳務自動對帳系統)

這是一個使用 Python (Pandas) 編寫的自動化工具，用於將網銀下載的 CSV 交易明細自動歸戶。
解決了原本需要人工逐筆核對匯款來源的繁瑣工作，將效率提升並減少人為錯誤。

## 🚀 功能特色
1. **支出自動過濾**：自動識別支出款項並排除，只針對存入款項進行分類。
2. **三層識別邏輯**：
   - **Level 1 (VIP優先)**：優先比對歷史資料庫 (History DB)，採「末六碼」比對，容錯率高。
   - **Level 2 (規則識別)**：支援 Regex 正規表達式，可設定萬用關鍵字 (如薪資、特定月份代號)。
   - **Level 3 (人工最後防線)**：無法識別的款項自動標記為「需人工確認」，並匯出獨立報表。
3. **自動學習**：使用者只需將人工確認後的結果填入 Excel，下次程式即可自動識別。

## 🛠️ 使用技術
- Python 3
- Pandas (資料處理)
- Openpyxl (Excel 讀寫)
- Regex (規則匹配)

## 📂 檔案結構說明
- `main.py`: 主程式碼
- `history.xlsx`: (需自行建立) 客戶對照資料庫
- `requirements.txt`: 依賴套件列表

## ⚠️ 注意事項
為了保護隱私，本倉庫不包含真實的財務數據 CSV 與客戶名單 Excel。
